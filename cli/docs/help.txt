── Commando Help ──────────────────────────────────────────────
Commando is the Official Reference CLI for AKKU Development.

Built-in commands:
  help              Show this message
  clear / cls       Clear the screen
  exit / quit       Exit commando
  history           Show plan history and perform manual rollback

Flags (cfg commands only — position-independent):
  --dry-run         Show the plan without saving or executing
  --force           Auto-approve without prompting (overrides --dry-run)

Usage pattern:
  <domain> <action> [target] [properties...] [flags]

Domain aliases:
  service / srv / services

Examples:
  service list
  service status apache2
  service cfg nginx running=true
  service cfg nginx running=true enabled=true --dry-run
  service cfg nginx masked=true --force
